# Datathon2026 - Access to care

This repository contains my data analysis project for Datathon 2026 hosted by DubsTech at the University of Washington. Using the NHIS Adult Summary Health Statistics dataset provided by CDC, I identify groups with limited healthcare access, explore healthecare access trends, and provide recommendations based on my findings. 

## Repository Contents
- `Datathon2026.ipynb` - Jupyter Notebook with full analysis and visualizations 
- `Datathon2026.html` - The HTML version of the analysis process and results for easy access
- `Access_to_Care_Dataset.csv` - The NHIS Adult Summary Health Statistics dataset provided 
- `Cleaned_Access_to_Care_Dataset.csv` - The cleaned dataset with attibutes and rows of interest 
- `Dashboard_by_gp.png` - Image of the dashboard showing percentage of individuals with limited healthcare access separated by demographic groups 
- `Dashboard_by_gp.png` - Image of the dashboard showing trends of subgroups for Employment status between 2019 and 2024

## Tools Used
- Python (Jupyter Notebook)
- Tableau
